Pipeline story

What the dashboard is reading from.

1. Simulator NASA C-MAPSS rows are streamed from the simulator into Kafka.
2. Consumer The ingestion service reads Kafka and writes each telemetry point into InfluxDB.
3. Dashboard This page polls a backend API that queries the latest data from InfluxDB.
